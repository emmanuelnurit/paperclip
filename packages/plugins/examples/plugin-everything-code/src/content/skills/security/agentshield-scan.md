# AgentShield Scan

- Defense in depth. No single control is enough.
- Validate inputs. Encode outputs.
- Treat external content as untrusted unless cryptographically attested.
- Rotate credentials on suspicion. Revoke on incident.
- Log enough to investigate, not enough to leak.
